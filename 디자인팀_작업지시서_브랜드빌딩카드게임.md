# 디자인팀 작업 지시서 — 브랜드 빌딩 카드 게임 in 스탬프투어

> 문서 버전 v1.0 · 작성: 기획 · 대상: 프로덕트/그래픽 디자인팀
> 목표: **광고주에게 ‘각인’과 ‘재미×효율’이 보이는 경험**을 설계한다.

---

## 0. 우리가 파는 것 (디자인 한 줄 미션)

이 서비스의 진짜 상품은 경품이 아니라 **광고주 브랜드의 ‘각인’**이다.
사용자가 브랜드명을 한 글자씩 직접 완성하는 동안 **풀네임이 늘 화면에 떠 있고, 채워지는 쾌감**을 느끼게 하라. 그 과정 자체가 광고주에게 가장 강력한 세일즈 포인트다.

디자인은 두 청중을 동시에 만족시켜야 한다.
- **사용자(관광객)** → 재미·기대감·수집 욕구 (도파밍)
- **광고주** → 브랜드가 정확히, 반복적으로, 매력적으로 노출된다는 확신

---

## 1. 디자인 원칙 (5)

1. **풀네임 상시 노출** — 미션 단어는 어느 화면에서도 브랜드 전체가 보이게. 절대 글자 조각만 강조하지 않는다.
2. **채워지는 쾌감** — 빈 칸 → 채운 칸 전이에 보상 모션을 준다. 진행감(progress)이 늘 보이게.
3. **기대감 연출** — 카드 뽑기는 ‘뒤집기 전’의 설렘이 핵심. 결과보다 과정의 두근거림.
4. **즐거운 맥락** — 여행·놀이의 밝은 톤. 광고가 아니라 게임처럼 느껴지게.
5. **브랜드 세이프티** — 광고주 브랜드는 정확한 철자·승인된 컬러/로고로만. (오타는 치명적 — 아래 5장)

## 2. 비주얼 아이덴티티

| 요소 | 가이드 |
|------|--------|
| 시그니처 모티프 | **빨간 발자국**(red footprint) — 포인트·아이콘·구분선에 일관 사용 |

| 역할 컬러 | 사용자/재미 = Teal `#1E7A60` · 광고주/효율 = Accent 빨강 (자료에서 양측을 대비) |

| 카드 | 흰 카드 + 라운드 + 그림자(뽑기 오브젝트), 빈 슬롯은 점선·회색 |

## 3. 핵심 화면 디자인 (앱)

### 3.1 미션 슬롯 (최우선 컴포넌트)
- 브랜드 풀네임을 칸으로 표현(예: `I O N I Q 5` 6칸).
- 상태 2종: **채움**(브랜드 컬러/진한 카드) · **빈칸**(회색 점선).
- 항상 풀네임 전체 노출. 보유 수량 배지는 보조 정보로 작게.
- 상단에 “미션 단어: {브랜드}” 라벨 — 철자 그대로.

### 3.2 빨간 발자국 획득 모달
- 빨간 발자국 아이콘 강조 + “빨간 발자국 획득” + ‘카드 선택하러 가기’ CTA.
- 보상 획득의 작은 축하 모션(스케일·발자국 찍힘).

### 3.3 카드 선택(럭키드로우) 모달
- 카드 5장 뒷면 제시 → 탭 → 뒤집힘(flip) → 글자 공개.
- 선택 카드는 강조(글로우), 나머지는 페이드/엑스 처리.
- 기대감 우선: 뒤집기 0.3–0.5s, 과하지 않게.

### 3.4 보관함 / 미션 진행
- 풀네임 슬롯 + 보유 알파벳 현황(글자별 수량) + 진행률.
- 완성 시 ‘선물 신청하기’ 버튼 활성(완성 전 비활성·이유 노출).

### 3.5 선물 신청 (선착순)
- **잔여 수량 강조**(“자동차 5대 중 2대 남음”) — 희소성·긴박감.
- 완성/마감/중복 상태별 명확한 피드백.

### 3.6 선물하기 (멀티)
- 친구 선택 → 글자 선택 → 전송. ‘함께 모은다’는 협력 감각을 시각적으로.

---

## 4. 모션 / 인터랙션 (도파밍 보상 설계)
- **칸 채워짐**: 글자 획득 시 해당 칸에 ‘톡’ 박히는 팝(scale + 살짝 회전).
- **카드 뽑기**: 뒷면→플립→공개의 3박자. 결과 전 0.2s 정지로 기대감.
- **완성 순간**: 풀네임 전체가 한 번 반짝/펄스 + 축하 — 브랜드가 ‘완성됐다’는 정서적 절정.
- 모든 모션은 `prefers-reduced-motion` 대응(끄기 제공). 1.5–2s 내, 과한 연출 금지.

---

## 5. 광고주 브랜드 세이프티 (필수 준수)
- **철자 정확성이 생명**: 미션 단어가 광고주 브랜드와 한 글자라도 다르면 ‘틀린 브랜드 각인’이 된다. (과거 IONIC5↔IONIQ5 같은 오타 절대 금지) — 디자인 QA 체크리스트에 ‘철자 대조’ 항목 포함.
- 광고주 CI(컬러·로고·서체)는 승인된 가이드 내에서만 사용. 임의 변형 금지.
- 풀네임 가독성 1순위(작은 화면에서도 또렷하게).

---

## 6. 광고주 세일즈 자료 디자인
앱 UI와 **같은 디자인 언어**로 제작(일관성 = 신뢰).
- **각인 데모 시각화**: 빈 슬롯이 한 글자씩 채워지는 과정을 정지/모션으로. (소개 페이지 히어로의 핵심)
- **재미 × 효율** 패널: 사용자=Teal / 광고주=빨강 대비, 가운데 ‘×’.
- **사례 카드**: 풀네임 완성 → 각인 흐름.
- 산출물: 소개 랜딩(웹) · 제안서(딜덱) · 한 장 요약(one-pager) 일관 템플릿.

---

## 7. 디자인 시스템 / 접근성
- 디자인 토큰(컬러·간격·라운드·타이포 스케일) 정의 → 개발 핸드오프.
- 명도 대비 WCAG AA 이상, 본문 최소 가독 크기 준수.
- 색맹 대응: 채움/빈칸을 색만이 아니라 **형태(점선 vs 실선)·명도**로도 구분.
- 라이트/다크 모드 모두 대응.

---
